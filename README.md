# ProiectTIC
ProiectTIC

Aplicatie -- AI ChatBot
Aplicatia este o aplicatie de tip chat cu un AI care ruleaza local si raspunde. Aplciatie faciliteaza comunicarea prin API intre utilizator si AI, salveaa conversatiile si permite accesarea lor utlerioara printr-un sistem de autentificare.
